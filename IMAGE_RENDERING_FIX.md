### 1. NumPy (.npy) —Ñ–∞–π–ª—ã

**–ü—Ä–æ–±–ª–µ–º–∞:** 
- NumPy Fortran-order –º–∞—Å—Å–∏–≤—ã –∏—Å–ø–æ–ª—å–∑—É—é—Ç **column-major** –ø–æ—Ä—è–¥–æ–∫
- HyperCube (–æ—Ç MATLAB) —Ç–∞–∫–∂–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç **column-major** –ø–æ—Ä—è–¥–æ–∫  
- –ö–æ–¥ –ø—ã—Ç–∞–ª—Å—è "–∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å" –∏–∑ Fortran –≤ C-order, –Ω–æ —ç—Ç–æ –ª–æ–º–∞–ª–æ –¥–∞–Ω–Ω—ã–µ!

**–†–µ—à–µ–Ω–∏–µ:**
–£–¥–∞–ª–µ–Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è `reorderFromFortran()` - –¥–∞–Ω–Ω—ã–µ —É–∂–µ –≤ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–º —Ñ–æ—Ä–º–∞—Ç–µ!

```swift

    values = reorderFromFortran(values, shape: header.shape)  // ‚ùå –≠—Ç–æ –ª–æ–º–∞–ª–æ –¥–∞–Ω–Ω—ã–µ!
}

// –°–¢–ê–õ–û (–ø—Ä–∞–≤–∏–ª—å–Ω–æ):
// –ü—Ä–æ—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º –¥–∞–Ω–Ω—ã–µ –∫–∞–∫ –µ—Å—Ç—å - –æ–Ω–∏ —É–∂–µ –≤ column-major!
```

**–ü–æ—á–µ–º—É —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç:**
- NumPy Fortran-order: `data[i0, i1, i2]` —Ö—Ä–∞–Ω–∏—Ç—Å—è –∫–∞–∫ `i0 + d0*(i1 + d1*i2)`
- MATLAB (matio): —Ç–∞–∫–æ–π –∂–µ –ø–æ—Ä—è–¥–æ–∫!
- –ù–µ –Ω—É–∂–Ω–∞ –∫–æ–Ω–≤–µ—Ä—Ç–∞—Ü–∏—è!

### 2. TIFF —Ñ–∞–π–ª—ã

**–ü—Ä–æ–±–ª–µ–º–∞ 1:** –ù–µ–ø—Ä–∞–≤–∏–ª—å–Ω—ã–π –ø–æ—Ä—è–¥–æ–∫ dims
- –ë—ã–ª–æ: dims = (C, H, W) –¥–ª—è CHW layout
- TIFF —ç—Ç–æ —Ñ–æ—Ä–º–∞—Ç –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏–π ‚Üí –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å (H, W, C) –¥–ª—è HWC layout

**–ü—Ä–æ–±–ª–µ–º–∞ 2:** –ù–µ–ø—Ä–∞–≤–∏–ª—å–Ω–∞—è –∏–Ω–¥–µ–∫—Å–∞—Ü–∏—è column-major
- TIFF —á–∏—Ç–∞–µ—Ç strips –≤ row-major: pixel[h,w] –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –∫–∞–Ω–∞–ª–∞
- –ù—É–∂–Ω–æ –∑–∞–ø–∏—Å–∞—Ç—å –≤ column-major –¥–ª—è MATLAB

**–†–µ—à–µ–Ω–∏–µ:**
–ò–∑–º–µ–Ω–µ–Ω –ø–æ—Ä—è–¥–æ–∫ dims –∏ —Ñ–æ—Ä–º—É–ª–∞ –∏–Ω–¥–µ–∫—Å–∞—Ü–∏–∏:

```c
// –ü—Ä–∞–≤–∏–ª—å–Ω—ã–π –ø–æ—Ä—è–¥–æ–∫ –¥–ª—è TIFF (–∫–∞–∫ –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏–µ)
outCube->dims[0] = H;  // 1526 (–≤—ã—Å–æ—Ç–∞)
outCube->dims[1] = W;  // 1536 (—à–∏—Ä–∏–Ω–∞)
outCube->dims[2] = C;  // 160 (–∫–∞–Ω–∞–ª—ã)

// –ü—Ä—è–º–∞—è –∑–∞–ø–∏—Å—å –≤ column-major (H, W, C)
// –î–ª—è pixel[row, col, channel]:

data[colMajorIdx] = pixel_value;
```

**Default layout:** CHW ‚Üí **HWC** (–ø—Ä–∞–≤–∏–ª—å–Ω–æ –¥–ª—è –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏–π)

**–í–∏–∑—É–∞–ª—å–Ω–æ:**

Row-major (TIFF):
```
[0,0] [0,1] [0,2] ... [0,W-1]
[1,0] [1,1] [1,2] ... [1,W-1]
...
```

Column-major (MATLAB/HyperCube):
```
[0,0] [1,0] [2,0] ... [H-1,0]
[0,1] [1,1] [2,1] ... [H-1,1]
...
```

## üìä –¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏–µ –¥–µ—Ç–∞–ª–∏

### Memory Layout

**Row-major (C order):**
```
–ò–Ω–¥–µ–∫—Å = i0 * d1 * d2 + i1 * d2 + i2
```
–ü–æ—Å–ª–µ–¥–Ω–∏–π –∏–Ω–¥–µ–∫—Å –∏–∑–º–µ–Ω—è–µ—Ç—Å—è –±—ã—Å—Ç—Ä–µ–µ –≤—Å–µ–≥–æ.

**Column-major (Fortran/MATLAB order):**
```
–ò–Ω–¥–µ–∫—Å = i0 + d0 * (i1 + d1 * i2)
```
–ü–µ—Ä–≤—ã–π –∏–Ω–¥–µ–∫—Å –∏–∑–º–µ–Ω—è–µ—Ç—Å—è –±—ã—Å—Ç—Ä–µ–µ –≤—Å–µ–≥–æ.
